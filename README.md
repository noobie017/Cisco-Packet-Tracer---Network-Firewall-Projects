# Cisco Packet Tracer - Network Firewall Projects

This repository contains multiple hands-on **firewall projects** built in Cisco Packet Tracer. 

Each project demonstrates different ways to implement firewall rules using both traditional router ACLs and the Cisco ASA firewall appliance.

### Projects Included

1. **Basic ACL Firewall** (`cisco-pt-basic-acl-firewall`)  
   Router acting as a basic firewall using Standard and Extended ACLs.

2. **ASA Basic Firewall** (`asa-basic-firewall`) ← **Current Project**  
   Cisco ASA 5505 configured with security levels, VLAN interfaces, and NAT/PAT.

(You can add more projects later: DMZ, Zone-Based Firewall, Site-to-Site VPN, etc.)

![ASA Basic Firewall Topology](screenshots/asa-basic-topology.png)

## 🎯 Learning Objectives

- Understand firewall fundamentals (stateful inspection, security levels, ACLs)
- Configure Cisco ASA 5505 with proper VLAN and interface setup
- Apply NAT/PAT for inside-to-outside communication
- Test traffic control between different security zones
- Practice real-world firewall verification commands
